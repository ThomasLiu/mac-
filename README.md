### 安装 [n](https://github.com/tj/n) 管理node.js 的版本
```
curl -L https://git.io/n-install | bash

. ~/.bashrc
```
使用n来安装最新的node.js
```
n latest
node -v
//检查安装后的node 的版本
npm -v 
//检查安装后的npm 的版本
```
